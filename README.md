# Agile Report

Read more about diagrams - https://timmson.github.io/fruit-report/

## Requirements

### As "classis"
    * Apache >= 2.2
    * PHP >= 5.3
    * SQLite >= 3

### As Docker container
    * Docker Compose

## Installation

### Get last sources
```
git clone https://github.com/timmson/fruit-report.git 
```

### Run
```
cd fruit-report/
docker-compose run -d
```
